Overview

DSL receiver for the block passed to #register. Builds a frozen list of child CMDx::Inputs. Supports arbitrary nesting: every DSL method accepts its own block.

def build(&)
  builder = new
  builder.instance_eval(&)
  builder.children.freeze
end

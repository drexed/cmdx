#call(value, options = {}) ⇒ Integer

Converts a value to an Integer

Examples:

Convert numeric strings to integers

Integer.call("42")      # => 42
Integer.call("-123")    # => -123
Integer.call("0")       # => 0

Convert numeric types to integers

Integer.call(42.0)      # => 42
Integer.call(3.14)      # => 3
Integer.call(0.0)       # => 0

Handle edge cases

Integer.call("")        # => 0
Integer.call(nil)       # => 0
Integer.call(false)     # => 0
Integer.call(true)      # => 1

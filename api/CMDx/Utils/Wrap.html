#array(object) ⇒ Array

Wraps an object in an array if it is not already an array.

Examples:

Already an array

Wrap.array([1, 2, 3])
# => [1, 2, 3]

Single value

Wrap.array(1)
# => [1]

Nil value

Wrap.array(nil)
# => []

def array(object)
  case object
  when Array then object
  when NilClass then EMPTY_ARRAY
  else Array(object)
  end
end
